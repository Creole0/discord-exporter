========================================
  使用流程
========================================

第一步：获取 Discord Bot Token
────────────────────────────────────
1. 访问 https://discord.com/developers/applications
2. 创建新应用（或选择已有应用）
3. 进入 "Bot" 页面
4. 点击 "Reset Token" 并复制
5. 将 Token 保存好（不要泄露！）

第二步：启动应用
────────────────────────────────────
双击 "启动.bat"，浏览器会自动打开

第三步：设置 Token
────────────────────────────────────
在网页的 "Bot Token 设置" 区域：
1. 粘贴你的 Token
2. 点击 "保存Token"
3. 看到 "已设置" 提示即成功

第四步：导出聊天记录
────────────────────────────────────
1. 在 Discord 中右键点击频道
2. 选择 "复制链接"
3. 粘贴到 "频道链接" 输入框
4. 选择时间范围（可选）
5. 选择导出格式
6. 点击 "开始导出"
7. 等待完成后点击下载

========================================
  常见问题
========================================

Q: 双击启动.bat 后闪退？
A: 右键点击 → "以管理员身份运行"

Q: 提示找不到 Python？
A: 双击运行 "安装Python.bat"，会自动安装

Q: 导出速度慢？
A: Discord API 有速率限制，这是正常的，请耐心等待

Q: 可以导出私信吗？
A: 目前仅支持服务器频道，不支持私信

Q: Token 安全吗？
A: Token 仅保存在本地内存中，不会上传到任何服务器
   建议定期更换 Token，不要分享给他人

Q: 如何停止应用？
A: 在命令窗口中按 Ctrl+C，或直接关闭窗口
