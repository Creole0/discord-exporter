# 更新日志

## 2026-03-02 - 时间选择优化

### 新增功能
- ✨ **精确时间选择**：支持选择具体的小时和分钟，不再仅限于日期
  - 开始时间和结束时间现在都可以精确到分钟
  - 前端使用 `datetime-local` 输入框替代原来的 `date` 输入框
  - 用户可以更精准地筛选特定时间段的聊天记录

### 改进
- 🔧 **向后兼容**：后端同时支持旧的日期格式和新的日期时间格式
  - 新格式：`YYYY-MM-DDTHH:MM` (如 `2026-03-02T14:30`)
  - 旧格式：`YYYY-MM-DD` (如 `2026-03-02`)，仍然可用
  - 使用旧格式时，结束日期会自动设置为当天的 23:59:59

### 用户体验
- 📝 更新了帮助文本，说明可以精确选择到具体小时和分钟
- 🎨 更新了标签文本，从"开始日期/结束日期"改为"开始时间/结束时间"

### 技术细节
- 前端：添加了 `input[type="datetime-local"]` 的 CSS 样式支持
- 后端：改进了日期时间解析逻辑，使用 try-except 实现格式兼容
- 错误处理：提供更详细的错误信息，便于调试

## 使用示例

### 精确时间筛选
- 导出 2026年3月2日 上午9点 到 下午5点 的消息
- 导出 2026年3月1日 晚上8点 到 3月2日 凌晨2点 的夜间聊天记录

### 留空选项
- 只设置开始时间：导出该时间之后的所有消息
- 只设置结束时间：导出该时间之前的所有消息
- 两个都留空：导出全部消息
